# WinUI3_APNG

Test control to play APNG files ([Animated PNG](https://www.w3.org/TR/png/#apng-frame-based-animation))
in WinUI 3 / C# with a [SwapChainPanel](https://learn.microsoft.com/en-us/windows/windows-app-sdk/api/winrt/microsoft.ui.xaml.controls.swapchainpanel?view=windows-app-sdk-1.8)

(tested on Windows 10 22H2, Windows App SDK 1.7.250401001)
